# Rilis v2.0.0

> ⚠️ **BREAKING CHANGE**: Template admin diubah sepenuhnya dari AdminKit ke **Tabler Admin**.
> Tidak ada kompatibilitas mundur pada tampilan frontend admin.

#### BREAKING CHANGES
1. Migrasi template admin dari AdminKit ke Tabler Admin secara penuh
2. Struktur layout navbar diubah ke "Combo Layout" Tabler (top + sub navbar)
3. Struktur footer diperbarui mengikuti gaya minimalis Tabler/Nginx Proxy Manager

#### FITUR BARU
1. Dark mode / Light mode toggle di navbar
2. Halaman index (Users, Roles, Permissions) menggunakan card layout gaya Tabler Employee table
3. Kolom pencarian terintegrasi di `card-header` sesuai template Tabler
4. Tombol aksi Edit & Delete langsung tampil inline di setiap baris tabel
5. Tombol "Permissions" dipulihkan di halaman Roles
